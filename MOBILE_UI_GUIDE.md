## Responsive Breakpoint Summary

### Tailwind Breakpoints Used

```css
/* Mobile-first approach */
sm:  640px  /* Small tablets */
md:  768px  /* Tablets */
lg:  1024px /* Desktop */
xl:  1280px /* Large desktop */
```

### Common Patterns

#### 1. Grid Transformations

```tsx
// 4 columns â†’ 2 â†’ 1
grid-cols-1 md:grid-cols-2 lg:grid-cols-4

// 3 columns â†’ 1
grid-cols-1 md:grid-cols-3

// 2 columns always
grid-cols-2 md:grid-cols-4
```

#### 2. Flex Direction

```tsx
// Stack vertically on mobile
flex-col sm:flex-row

// Reverse on mobile
flex-col-reverse lg:flex-row
```

#### 3. Visibility Toggles

```tsx
// Hide on mobile, show on desktop
hidden lg:block

// Show on mobile, hide on desktop
lg:hidden
```

#### 4. Spacing Adjustments

```tsx
// Smaller gaps on mobile
gap-3 lg:gap-6

// Reduced padding on mobile
p-4 lg:p-8

// Tighter margins on mobile
mb-4 lg:mb-8
```

#### 5. Text Sizing

```tsx
// Smaller text on mobile
text-sm lg:text-base
text-base lg:text-lg

// Icon sizing
w-4 h-4 lg:w-5 lg:h-5
```

## Touch-Friendly Enhancements

### Button Minimum Sizes

```tsx
// Minimum 44x44px touch target
className = 'px-3 py-2'; // Results in ~44px height

// Full-width on mobile
className = 'w-full sm:w-auto';
```

### Icon Buttons

```tsx
// Adequate padding for touch
<button className="p-2">
  {' '}
  // 8px padding + 16px icon = 32px min
  <Icon className="w-4 h-4" />
</button>
```

### Form Inputs

```tsx
// Comfortable input height
className = 'px-4 py-2'; // ~40px height
```

## Chart Mobile Patterns

### Horizontal Scroll Container

```tsx
<div className="overflow-x-auto -mx-4 px-4 lg:mx-0 lg:px-0">
  <div className="min-w-[300px]">
    <ResponsiveContainer width="100%" height={250}>
      <BarChart>...</BarChart>
    </ResponsiveContainer>
  </div>
</div>
```

**Why this works:**

- `-mx-4 px-4`: Extends scroll area to screen edges
- `lg:mx-0 lg:px-0`: Removes on desktop
- `min-w-[300px]`: Prevents chart collapse
- Smaller height on mobile saves vertical space

### Chart Text Sizing

```tsx
// Mobile-friendly font sizes
tick={{ fill: '#9CA3AF', fontSize: 10 }} // Mobile
tick={{ fill: '#9CA3AF', fontSize: 12 }} // Desktop (via className)

// Narrower Y-axis on mobile
width={40} // Mobile
width={60} // Desktop (default)
```

## Quick Fixes for Common Issues

### Chart Not Scrolling

```tsx
// âŒ Wrong
<div className="overflow-x-auto">
  <ResponsiveContainer>...</ResponsiveContainer>
</div>

// âœ… Correct
<div className="overflow-x-auto">
  <div className="min-w-[300px]"> {/* Add wrapper */}
    <ResponsiveContainer>...</ResponsiveContainer>
  </div>
</div>
```

### Text Overflow

```tsx
// âŒ Wrong
<p className="text-sm">{longText}</p>

// âœ… Correct
<p className="text-sm truncate">{longText}</p>
// or
<p className="text-sm line-clamp-2">{longText}</p>
```

### Buttons Too Small

```tsx
// âŒ Wrong
<button className="px-2 py-1">

// âœ… Correct
<button className="px-3 py-2 lg:px-4"> {/* Minimum 44px */}
```

### Cards Not Stacking

```tsx
// âŒ Wrong
<div className="flex gap-4">

// âœ… Correct
<div className="flex flex-col lg:flex-row gap-4">
```
